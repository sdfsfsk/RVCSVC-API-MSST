# MSST model directory

Place supported vocal-separation weights here:

```text
model_bs_roformer_ep_317_sdr_12.9755.ckpt
bs_roformer_karaoke_frazer_becruily.ckpt
```

The matching YAML files are included under `msst/configs/vocal_models/`. Obtain weights from their authors or a source recognized by [Music-Source-Separation-Training](https://github.com/ZFTurbo/Music-Source-Separation-Training). Model weights are intentionally excluded from this repository.
